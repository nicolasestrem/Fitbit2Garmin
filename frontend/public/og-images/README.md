# OpenGraph Images

This directory contains OpenGraph (OG) images for social media sharing.

## Required Image

**`default-og.png`** - Default fallback image (1200x630px)
- Used for all pages until specific images are created
- Should include:
  - TrackerSync branding
  - Simple visual representation of the conversion (Fitbit â†’ Garmin)
  - Clean, professional design
  - Readable text even at thumbnail sizes

## Image Specifications

- **Dimensions:** 1200x630px (required for Facebook/LinkedIn/Twitter)
- **Format:** PNG or JPG
- **File size:** < 1MB for fast loading
- **Content:** Should not include text that's critical to understanding (text may be cut off on some platforms)

## Design Tools

Recommended tools for creating OG images:
- Figma (professional, free tier available)
- Canva (easy templates)
- Adobe Photoshop
- GIMP (free alternative)

## Testing

Test your OG images using:
- Facebook Sharing Debugger: https://developers.facebook.com/tools/debug/
- Twitter Card Validator: https://cards-dev.twitter.com/validator
- LinkedIn Post Inspector: https://www.linkedin.com/post-inspector/
